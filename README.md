# React-Todo-App
 A todolist app built with React.

### Features
+ Beautiful UI
+ Full CRUD for todos
+ Local Storage used to store todos data

##### Built With
+ HTML5
+ CSS3
+ React

### Getting Started
clone repo

`cd` into folder and run `npm install`

run `npm start`
